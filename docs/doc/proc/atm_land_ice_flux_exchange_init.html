public subroutine atm_land_ice_flux_exchange_init(Time, Atm, Land, Ice, atmos_ice_boundary, land_ice_atmos_boundary, Dt_atm_in, Dt_cpl_in, z_ref_heat_in, z_ref_mom_in, do_area_weighted_flux_in, do_forecast_in, partition_fprec_from_lprec_in, scale_precip_2d_in, nblocks_in, cplClock_in, ex_gas_fields_atm_in, ex_gas_fields_ice_in, ex_gas_fluxes_in)

Subroutine to initialize FMS modules, diagnostics and boundary arrays

The following module variables are initialized:
1. tr_table and tr_table_map to map tracer indices between components code
2. frac_precip if scale_precip_2d is true code
3. exchange grid indices X1_GRID_ATM, X1_GRID_ICE, and X1_GRID_LND code
4. ni_atm, nj_atm code
5. nxc_ice and nyc_ice code
6. nxc_lnd and nyc_lnd code
7. clocks code

The following derived types are initialized 1. atmos_ice_boundary code
2. gases fluxes in Ice%ocean_fields code
3. land_ice_atmos_boundary code
4. addtional as fields in Atm%fields code

The following FMS subroutines are initialized 1. diag_integral_mod code
2. fms_diag_integral_field_init for each field example code
